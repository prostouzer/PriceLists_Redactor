"Редактор прайс-листов" - приложение на ASP.NET MVC Web Api с исп. JQuery, Bootstrap.</br>
Взаимодействие с базой данных через EntityFramework.</br>
Для редактирования отдельных ячеек прямо на таблице (*прайс-лист* => "Подробнее") исп. X-editable.</br>
Интерактивные таблицы сделаны с исп. DataTables.</br>
</br>
Для тестирования исп. NUnit + тестовая копия контекста базы данных.
