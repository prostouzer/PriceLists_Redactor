Редактор прайс-листов - приложение на ASP.NET MVC Web Api с исп. JQuery, Bootstrap.
Взаимодействие с базой данных через EntityFramework.
Для редактирования отдельных ячеек прямо на таблице (*прайс-лист* => "Подробнее") исп. X-editable.
Интерактивные таблицы сделаны с исп. DataTables.

Для тестирования исп. NUnit + тестовая копия контекста базы данных.
